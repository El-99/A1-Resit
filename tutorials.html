HTML Tutorial (For beginners):

Most of the stuff on the web is no different than the stuff on your computer — it’s just a whole load of files sorted into a whole load of directories. HTML files are nothing more than simple text files, so to start writing in HTML, you need nothing more than a simple text editor.

The site we’ll be using for most of the content in this tutorial will be htmldog.com, be sure to visit the site as there is more resources available. The primary thing to keep in mind, the supermagic key, is that HTML is used for meaning and CSS is used for presentation. HTML is nothing more than fancy structured content and the visual formatting of that content will come later when we tackle CSS. You might find different approaches elsewhere on the web but HTML Dog focuses on best practice from the outset and getting into the frame of mind of doing things the right way from the start will lead to much better results in the end.

CSS Tutorial:

In the event that you are now comfortable and confident with the structure of your website. It is essential to add some styling sheets to it to provide good presentational layout. CSS exists to style your HTML—to present your content.The language of Cascading Style Sheets is somewhat different to HTML but it remains simple and straightforward.

The basic structure of every web page, HTML, is very plain on its own. The beautiful websites that you see across the internet are styled with a variety of tools, including CSS. CSS, or Cascading Style Sheets, is a language that web developers use to style the HTML content on a web page. If you’re interested in modifying colors, font types, font sizes, shadows, images, element positioning, and more, CSS is the tool for the job!

User Interace Design:

For the user interface user design, we will focus on the material Design of the website. Material Design is a visual language that synthesizes the classic principles of good design with the innovation of technology and science.

Material Design is inspired by the physical world and its textures, including how they reflect light and cast shadows. Material surfaces reimagine the mediums of paper and ink.
